
TS脚手架工具：TypeScript library starter
  使用：
    git clone https://github.com/alexjoverm/typescript-library-starter.git ts-axios
    cd ts-axios
    npm install
    src目录下建一个与自己重命名一致的.ts文件作为入口文件

npm start 打包
npm run lint 代码预发检测
npm test 单元测试
